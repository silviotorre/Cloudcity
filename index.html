Cloud city 1th edition

Time ago, in a cloud far away...

Innovation cost a lot of money, much more in a lack of standards. Interoperability is also a crucial factor, and security too.

In 2013 Silvio Torre and Claudio Quaranta did a one-off conference about this in Orvieto, Italy, talking about technology, design ed the necessary vision to step forward the smartcity paradigm.

"We hope in few year to see great number of smart cities coming up with their services and urban network ecosystems, but nobody want a great number of apps and login for all these cities. " (S. Torre)

Maybe too early but Cloudcity was born.
Now is time to build a federative system, in a solid framework, accessible and lightning fast as the user needs.

Cloudcity, a smart cities interoperability framework

The Framework

Cloudcity is the response needed to interoperability and front end accessibility and convergence for our smart-cities constellation.

Is a solid Java based micro-services architecture, developed in a Model View Controller (MVC) pattern, with a state of the art front end react js "ready to customize " code.

And is modular.
